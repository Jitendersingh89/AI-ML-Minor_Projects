# Sentiment Analysis using RNN
This project implements a **Sentiment Analysis** model using a **Recurrent Neural Network (RNN)** in PyTorch. The model classifies text reviews into **Positive** and **Negative** sentiments.

## Dataset
- IMDb Movie Reviews Dataset
- Binary Classification
  - Positive Reviews
  - Negative Reviews
 
 ## Project Structure
Sentiment_Analysis/
─ Sentiment_Analyisi.ipynb
─ requirements.txt
─ README.md

## Tech Stack

- Python
- PyTorch
- NumPy
- Pandas
- Scikit-learn
- Jupyter Notebook

## Features

- Text preprocessing
- Tokenization and vectorizationn
- RNN-based binary classifier
- Model training using PyTorch
- Model evaluation on test data
- Binary sentiment prediction

## Model Architecture
- Embedding Layer
- Simple RNN Layer
- Fully Connected Layer
- Sigmoid Activation

Loss Function:
- BCELoss(Binary CrossEntropy Loss)

Optimizer:
- Adam
